# 阿里云邮箱授权码生成指南

## 什么是授权码？

授权码是用于第三方客户端（如 Python 脚本、邮件客户端）登录邮箱的专用密码，**不是你的登录密码**。使用授权码可以：
- 避免泄露主密码
- 随时单独撤销授权
- 支持两步验证的邮箱

---

## 📋 生成步骤

### 1. 登录阿里云邮箱

访问：https://qiye.aliyun.com/ (企业邮箱) 或 https://mail.aliyun.com/ (个人邮箱)

### 2. 进入设置

```
右上角齿轮图标 ⚙️ → 邮箱设置 → 账户安全
```

### 3. 开启 IMAP/SMTP 服务

找到 **"客户端设置"** 或 **"POP3/IMAP/SMTP"** 选项：

- ✅ 开启 **IMAP/SMTP 服务**
- ✅ 开启 **客户端授权码**（如果需要）

### 4. 生成授权码

1. 点击 **"生成授权码"** 或 **"管理授权码"**
2. 可能需要短信验证
3. 设置授权码名称（如："OpenClaw 邮件总结"）
4. 系统会生成一串 16 位随机字符

### 5. 保存授权码

**重要**：授权码只显示一次！立即复制到安全的地方：
- 1Password
- 或其他密码管理器

---

## ⚠️ 注意事项

1. **授权码 ≠ 登录密码** - 不要混淆
2. **只显示一次** - 丢失需要重新生成
3. **可以撤销** - 在"管理授权码"里可以删除旧的
4. **一个设备一个码** - 建议为每个用途生成独立授权码
5. **定期更换** - 建议每 3-6 个月更换一次

---

## 🔒 安全建议

- 将授权码存入 1Password
- 不要直接发在聊天里（用私密方式）
- 为 OpenClaw 单独生成一个授权码
- 定期检查授权码使用记录
