### 摄像头调用方式 ⭐
**问题**：误以为 `nodes camera_snap` 可以调用 USB 摄像头，导致失忆

**解决方案**：
- macOS USB 摄像头使用 `imagesnap` CLI 调用
- 命令：`imagesnap -d "Insta360 Link 2" -q high [输出文件.jpg]`
- 查看设备：`imagesnap -l`
- `nodes camera_snap` 仅适用于配对的节点设备（手机等）

**设备信息**：
- 主摄像头：Insta360 Link 2 (USB 直连)
- 备选：Insta360 Virtual Camera, "Daler's iPhone"的相机
- imagesnap 位置：`/opt/homebrew/bin/imagesnap`

- `workspace/TOOLS.md` - 详细配置
- `workspace/MEMORY.md` - 长期记忆
